# Homebrew Tap for fns-cli

Official [Homebrew](https://brew.sh) tap for [fns-cli](https://fns-cli.afonso.dev) — a CLI tool designed to assist with daily developer tasks.

## Installation

```bash
brew install afonsodemori/tap/fns-cli
```

## Upgrade

```bash
brew upgrade fns-cli
```

## Uninstall

```bash
brew uninstall fns-cli
```

## About

`fns-cli` is a Go-based CLI tool designed to assist with daily developer tasks, specifically focusing on integrations with Jira and GitLab.

- [Documentation](https://fns-cli.afonso.dev/guide)
- [GitHub](https://github.com/afonsodemori/fns-cli)
